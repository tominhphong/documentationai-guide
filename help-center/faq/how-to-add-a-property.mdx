## Overview

You can quickly add a property listing in Realtor OS through the dashboard.

## Steps

1. Log in to your Realtor OS account.
2. Click on "Properties" in the main menu.
3. Select "Add New Property".
4. Enter property details: address, price, description.
5. Upload images (optional).
6. Click "Save" to publish the listing.

<Callout kind="info">
You can edit property info anytime after saving.
</Callout>
