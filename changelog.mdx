## How to Upgrade

Follow these steps to upgrade Realtor OS safely. Always back up your data first.

<Callout kind="info">
  Test upgrades in a staging environment before production deployment.
</Callout>

<Steps>
  <Step title="Backup Data" icon="database">
    Export your listings and configurations:
    
````bash
realtor-os export --backup-dir=/path/to/backup
````
  </Step>
  
  <Step title="Update Package" icon="package">
    Use your preferred package manager:
    
    <CodeGroup tabs="npm,yarn,pnpm">
    ````bash
npm install realtor-os@latest
````
    
    ````bash
yarn add realtor-os@latest
````
    
    ````bash
pnpm add realtor-os@latest
````
    </CodeGroup>
  </Step>
  
  <Step title="Run Migrations" icon="git-branch">
    
````bash
realtor-os migrate --env=production
````
    
    Review migration logs for any issues.
  </Step>
  
  <Step title="Verify" icon="check-circle">
    Test key endpoints like property search at `https://api.example.com/v2/properties`.
  </Step>
</Steps>
