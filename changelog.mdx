## 2026-04-26 — Site Mode: Public

Switched from Private+OAuth → Public.

**Reason:**
- Documentation.AI MCP `/_mcp` only serves public docs by default (per `/docs/ai/mcp-server`)
- OAuth had a `Session Expired (400)` bug at the callback step we couldn't work around (state cookie validates correctly via curl but browser flow fails)

**Verification:** 5/5 sample MCP queries return correct content from new pages. Site root now serves HTTP 200 directly (no OAuth redirect).
