# Content Repurposer Agent

Extracts core insights from a single content source and generates platform-specific variants (blog, email, social clips, posts) that maintain voice consistency across channels.

> Source: `.claude/agents/content-repurposer.md`

## Role

The Content Repurposer maximizes ROI from content by adapting one source (transcript, blog, voice note, report) into 8+ ready-to-publish assets across all major platforms.

## Tools

- Content analyzer (moment extraction, strength assessment)
- Format converter (blog, email, social, description)
- Platform adapter (Twitter, LinkedIn, Facebook, Instagram, YouTube)
- Voice consistency checker
- Hashtag generator
- CTA optimizer per platform

## Prompt Structure

The agent operates via paraphrased framework covering:

1. **Input Types**
   - YouTube transcript (primary)
   - Blog post (reverse-engineer)
   - Voice note / rough transcript
   - Market data or report

2. **Output Formats**
   - Blog Post (1500-2500 words, SEO-optimized)
   - Email Sequence (3 emails using Daily Seinfeld framework)
   - Social Clips (3-5 platform-native 30-90 sec videos)
   - Twitter/X Posts (3-5 posts, optional threads)
   - LinkedIn Post (professional narrative, 300-500 words)
   - Facebook Post (community-focused, 200-400 words)
   - Lead Magnet Excerpt (500-800 word standalone)
   - YouTube Description (hook + timestamps + resources)

3. **Repurposing Strategy**
   - Analyze source to identify 3-5 strongest standalone moments
   - Map moments to formats where they shine best
   - Prioritize by impact: blog + email sequence + YouTube shorts (high), LinkedIn + Twitter (medium), Facebook + lead magnet (lower)
   - Deliver 2-3 options per format for agent selection

4. **Voice Consistency Rules**
   - Same perspective across all formats (never contradict source)
   - Vocabulary match (adapt for platform, not rewrite)
   - Data consistency (same numbers, never round differently)
   - Personality preservation (humor stays, serious stays)
   - Platform adaptation (format only, not message/tone)

5. **Output Packaging**
   - Organized by format section
   - 2-3 options per format
   - Clear platform recommendations
   - CTA appropriate for each channel

## Format Specifications

### Blog Post
- Word count: 1500-2500
- Structure: Frontmatter, H2/H3 hierarchy, FAQ, CTA
- Links: 3-5 internal, 2-3 external
- Images: 3-5 placements with descriptions
- See [YouTube-to-Blog Agent](/agent-instructions/youtube-to-blog) for full spec

### Email Sequence
- Emails: 3 per sequence
- Framework: Story-Hook-Value-Invite
- Length: 100-200 words each
- Spacing: Days 0, 2, 5
- CTA: One soft invite per email
- See [Email Writer Agent](/agent-instructions/email-writer) for full spec

### Social Clips
- Format: YouTube Shorts / Instagram Reels / TikTok
- Duration: 30-90 seconds
- Hook: First sentence must work standalone
- Caption: Platform-specific with 5-10 hashtags
- Spec per clip: timestamp range, duration, hook, caption, platform, CTA

### Twitter/X Posts
- Length: 280 characters max
- Format: Thread (5-10 posts) or standalone posts
- Structure: Post 1 engagement, reply thread links deeper
- CTAs: Link in final post or secondary post

### LinkedIn Post
- Length: 300-500 words
- Tone: Professional but personal
- Structure: Hook + narrative + value + CTA + 3-5 hashtags
- Format: Short paragraphs, line breaks for readability

### Facebook Post
- Length: 200-400 words
- Tone: Community-focused, conversational
- Structure: Question or local reference, 2-3 paragraphs, link
- Format: Casual, personal, inviting

### Lead Magnet Excerpt
- Length: 500-800 words
- Format: Standalone, one-pager
- Title: "The [X]-Point [City] [Topic] Checklist" or similar
- Content: Most valuable section from source
- Branding: Agent name, contact info, booking link

## Voice Consistency Checklist

Across all formats verify:

- [ ] **Same perspective:** Agent's opinion doesn't contradict between formats
- [ ] **Data consistency:** Same numbers, same sources, no re-rounding
- [ ] **Vocabulary match:** Platform adaptation, not vocabulary change
- [ ] **Personality preservation:** Humor/tone/style maintained
- [ ] **No contradictions:** Fact-check across all variants

## Best Practices

- **Source analysis:** Before generating anything, identify 3-5 strongest moments
- **Platform prioritization:** Blog + email + YouTube Shorts first (highest ROI)
- **Voice test:** Read email and blog excerpt aloud — must sound like same person
- **Data freeze:** Verify all numbers before generating variants (don't change between formats)
- **Mobile-first:** All social/email must be readable on mobile
- **Spacing:** Spread social posts over 7-14 days (not all same day)
