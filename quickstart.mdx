# Quickstart

End-to-end setup of the upstream `claude-code-real-estate-framework` repo. Adapted from `README.md` + `DESKTOP-SETUP.md` of the upstream repo. For the deep setup guide, see Help Center → Setup Guide (Phase 2 — coming).

## Prerequisites

| Requirement | Why |
|---|---|
| macOS or Linux | Setup scripts assume Unix |
| Homebrew | Installs Python, Node, yt-dlp, ffmpeg |
| Claude Code CLI | Runs the agents + MCP wrapper |
| GitHub account | Clone the upstream repo |

## Step 1 — Install System Dependencies

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install python@3.12 node yt-dlp ffmpeg
```

What each does:
- **Python 3.12** — Backend tools (RAG, CRM, CSV, content gen)
- **Node** — MCP server bridge (TypeScript)
- **yt-dlp** — Free YouTube transcript downloader
- **ffmpeg** — Audio/video processing

## Step 2 — Install Claude Code

```bash
npm install -g @anthropic-ai/claude-code
claude  # follow auth prompts
```

## Step 3 — Clone the Upstream Repo

```bash
cd ~/
git clone https://github.com/puzzled-mushroom911/claude-code-real-estate-framework.git real-estate-ai
cd real-estate-ai
```

## Step 4 — Run Setup Script

```bash
bash scripts/setup.sh
```

Installs Python deps, builds MCP server (TypeScript → JS), creates manifest registry at `~/.real_estate_ai/`. Takes 2-5 minutes.

## Step 5 — Configure Agent Profile

```bash
cp config/.env.example config/.env
```

Edit `config/agent_profile.yaml`:

```yaml
agent:
  name: "Your Name"
  phone: "(555) 123-4567"
  email: "you@example.com"
  brokerage: "Your Brokerage"
  city: "Your City"
  state: "TX"
  youtube_handle: "@yourchannel"
  booking_link: "https://calendly.com/you"
```

## Step 6 — Add Optional API Keys

Edit `config/.env`:

| Key | Purpose | Required? |
|---|---|---|
| `CRM_API_KEY` | GoHighLevel CRM integration | Only if using GHL |
| `CRM_LOCATION_ID` | GHL location | Only if using GHL |
| `ANTHROPIC_API_KEY` | Enhanced guide generation | Optional |

Core features (RAG, content, marketing) work **without** any API keys.

## Step 7 — Wire Up MCP Server

Add to `~/.mcp.json`:

```json
{
  "mcpServers": {
    "real-estate-agent": {
      "command": "node",
      "args": ["/Users/YOU/real-estate-ai/mcp_server/dist/index.js"]
    }
  }
}
```

Replace path with absolute path to your clone.

## Step 8 — Restart Claude Desktop / Code

After restart, MCP tools appear under `mcp__real-estate-agent__*` namespace. Verify by asking Claude: *"List available MCP tools."*

## Step 9 — First Query (smoke test)

In Claude Code:

```
Use the rag_list_databases tool.
```

If empty list returned, MCP is working. Next: ingest content via `youtube_fetch_videos` or build a knowledge base from PDFs in `knowledge_bases/documents/`.

## Common Issues

- **MCP not loading** → check Claude Desktop logs at `~/Library/Logs/Claude/`
- **Python deps fail** → ensure `python@3.12` is on PATH (check `which python3`)
- **yt-dlp blocked** → update: `brew upgrade yt-dlp`

Full troubleshooting: Help Center → Troubleshooting (Phase 2 — coming).
