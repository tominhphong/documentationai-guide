# Python Conventions

> Source: `.claude/rules/python-conventions.md`

This guide defines coding standards for Python scripts in `tools/` and any data processing scripts. Follow these conventions when adding or extending tools.

## Style & Naming

### Naming Conventions

| Type | Convention | Example |
|------|-----------|---------|
| **Files** | snake_case | `create_knowledge_base.py`, `search_contacts.py` |
| **Functions** | snake_case | `fetch_transcript()`, `search_knowledge_base()` |
| **Variables** | snake_case | `user_input`, `vector_store`, `max_results` |
| **Constants** | UPPER_SNAKE_CASE | `MAX_RETRIES`, `API_TIMEOUT_SECONDS`, `DEFAULT_CHUNK_SIZE` |
| **Classes** | PascalCase | `YouTubeTranscriptFetcher`, `KnowledgeBaseManager` |

### Main Guard Pattern

Every script must use the `if __name__ == "__main__":` guard:

**✓ Correct:**
```python
def process_contacts(csv_path: str) -> list[dict]:
    """Load and parse contacts from CSV."""
    # implementation
    pass

if __name__ == "__main__":
    import sys
    if len(sys.argv) < 2:
        print("Usage: python process_contacts.py <csv_path>")
        sys.exit(1)
    
    contacts = process_contacts(sys.argv[1])
    print(f"Processed {len(contacts)} contacts")
```

**✗ Avoid:**
```python
# Code runs immediately on import, breaking reusability
contacts = process_contacts("data.csv")
print(f"Processed {len(contacts)} contacts")
```

### Type Hints on All Functions

Every function signature must include type hints:

**✓ Correct:**
```python
def search_knowledge_base(
    query: str,
    limit: int = 10
) -> list[dict]:
    """Search vector database for query.
    
    Args:
        query: Search string
        limit: Max results to return
        
    Returns:
        List of matching documents with scores
    """
    # implementation
    pass
```

**✗ Avoid:**
```python
def search_knowledge_base(query, limit=10):
    # No type hints
    pass
```

### Docstrings (Google Style)

Public functions must have docstrings:

**✓ Correct:**
```python

    """Download and index transcripts from a YouTube channel.

            - chunks_created (int)
```

## Project Structure

### tools/ Organization

Each subdirectory contains standalone scripts, NOT a monolithic package:

```
tools/
├── rag_tools/
│   ├── create_knowledge_base.py
│   ├── search_knowledge_base.py
│   ├── ingest_youtube_channel.py
│   └── requirements.txt             # Dependencies specific to this tool group
├── youtube_tools/
│   ├── download_transcripts.py
│   ├── extract_metadata.py
│   └── requirements.txt
├── content_tools/
│   ├── generate_blog_post.py
│   ├── generate_email_sequence.py
│   └── requirements.txt
├── csv_tools/
├── guide_tools/
├── crm_tools/
├── utilities/
│   ├── logger.py                    # Shared logging setup
│   ├── config_loader.py             # Load env vars and config
│   └── validators.py                # Common validation functions
└── requirements.txt                 # All dependencies
```

**Philosophy:** Scripts are callable independently, not imported from each other (minimal coupling). Shared functions go in `utilities/`.

### Configuration & Secrets

**Use environment variables, not hardcoded values:**

**✓ Correct:**
```python
import os
from dotenv import load_dotenv

load_dotenv()  # Load from .env file

crm_api_key = os.getenv("CRM_API_KEY")
if not crm_api_key:
    raise ValueError("CRM_API_KEY not set in .env")

client = CRMClient(api_key=crm_api_key)
```

**✗ Avoid:**
```python
# Hardcoded credentials in source code
client = CRMClient(api_key="ghl-abc123xyz")
```

**.env File (never committed):**
```bash
CRM_API_KEY=your-key-here
ANTHROPIC_API_KEY=sk-xxx
SUPABASE_URL=https://project.supabase.co
SUPABASE_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
```

### File Operations with pathlib

Use `pathlib.Path` instead of `os.path`:

**✓ Correct:**
```python
from pathlib import Path

def save_results(output_dir: str, filename: str) -> Path:
    """Save results to a file."""
    output_path = Path(output_dir) / filename
    output_path.parent.mkdir(parents=True, exist_ok=True)
    output_path.write_text(json.dumps(results))
    return output_path
```

**✗ Avoid:**
```python
import os

def save_results(output_dir, filename):
    output_path = os.path.join(output_dir, filename)
    os.makedirs(os.path.dirname(output_path), exist_ok=True)
    # ...
```

Scripts with CLI interfaces must use `argparse`:

**✓ Correct:**
```python
import argparse

        "channel_handle",
```


## Error Handling

### Use Specific Exception Types

**✓ Correct:**
```python
def search_knowledge_base(query: str) -> list[dict]:
    """Search vector database."""
    try:
        results = vector_store.search(query)
        return results
    except ValueError as e:
        # Handle validation errors (e.g., empty query)
        logger.error(f"Invalid search query: {e}")
        raise
    except Exception as e:
        # Log unexpected errors with context
        logger.exception(f"Unexpected error searching knowledge base: {e}")
        raise RuntimeError(f"Search failed: {e}") from e
```

**✗ Avoid:**
```python
def search_knowledge_base(query):
    try:
        return vector_store.search(query)
    except:  # Bare except catches everything, hides bugs
        pass
```

### Logging Instead of Print

Use the `logging` module for production code:

**✓ Correct:**
```python
import logging

logger = logging.getLogger(__name__)

def ingest_channel(channel_handle: str) -> dict:
    logger.info(f"Starting ingestion for channel: {channel_handle}")
    
    try:
        transcripts = download_transcripts(channel_handle)
        logger.info(f"Downloaded {len(transcripts)} transcripts")
        
        results = create_knowledge_base(transcripts)
        logger.info(f"Created {results['chunks_created']} vector chunks")
        return results
        
    except Exception as e:
        logger.error(f"Ingestion failed: {e}", exc_info=True)
        raise
```

**✗ Avoid:**
```python
print("Starting ingestion...")
print(f"Downloaded {len(transcripts)} transcripts")  # Not captured for debugging
```

## Structured Results

Return structured data (dict or dataclass), not raw strings:

**✓ Correct:**
```python
from dataclasses import dataclass

@dataclass
class SearchResult:
    query: str
    matches: list[dict]
    total_found: int
    confidence_scores: list[float]
    search_time_ms: float

def search_knowledge_base(query: str) -> SearchResult:
    """Search and return structured result."""
    start_time = time.time()
    matches = vector_store.search(query)
    elapsed = (time.time() - start_time) * 1000
    
    return SearchResult(
        query=query,
        matches=matches,
        total_found=len(matches),
        confidence_scores=[m["score"] for m in matches],
        search_time_ms=elapsed
    )
```

**✗ Avoid:**
```python
def search_knowledge_base(query):
    matches = vector_store.search(query)
    return f"Found {len(matches)} results for '{query}'"  # String is hard to parse
```

## Testing

### Test File Location & Naming

Tests live alongside source files with `test_` prefix:

```
tools/
├── rag_tools/
│   ├── create_knowledge_base.py
│   ├── test_create_knowledge_base.py    # Tests for this module
│   ├── search_knowledge_base.py
│   └── test_search_knowledge_base.py
```

### Using pytest

**✓ Correct:**
```python
import pytest
from create_knowledge_base import ingest_documents

def test_ingest_documents_with_valid_path(tmp_path):
    """Test successful document ingestion."""
    # Create a test document
    doc_path = tmp_path / "test.txt"
    doc_path.write_text("Sample content")
    
    result = ingest_documents(str(tmp_path))
    
    assert result["success"] is True
    assert result["chunks_created"] > 0

def test_ingest_documents_with_invalid_path():
    """Test error handling for missing directory."""
    with pytest.raises(FileNotFoundError):
        ingest_documents("/nonexistent/path")

def test_search_returns_scored_results(mock_vector_store):
    """Test search result structure."""
    results = search_knowledge_base("test query")
    
    assert isinstance(results, SearchResult)
    assert results.query == "test query"
    assert all(0 <= score <= 1 for score in results.confidence_scores)
```

**Run tests:**
```bash
pytest                         # Run all tests
pytest -v                      # Verbose output
pytest --cov tools/            # With coverage report
pytest tests/rag_tools/        # Specific directory
```
