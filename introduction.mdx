# What This System Is

`realtor-os-wiki` is a **private technical reference** documenting the open-source repo [`puzzled-mushroom911/claude-code-real-estate-framework`](https://github.com/puzzled-mushroom911/claude-code-real-estate-framework) AS-IS.

The upstream framework is an AI-powered toolkit for real estate agents using Claude Code: YouTube-to-knowledge-base, content repurposing, lead nurture emails, CRM automation, and relocation guides. It ships **9 agents**, **17 MCP tools**, and marketing playbooks distilled from Hormozi, Brunson, and Channel Junkies.

## What This Wiki Documents

- **9 agents** — system prompts for email-writer, customer-response, youtube-pipeline, teleprompter, voice-processor, content-repurposer, relocation-guide-generator, crm-agent, youtube-to-blog
- **6 MCP tool groups** — `rag`, `youtube`, `content`, `crm`, `csv`, `guide` (17 tools total)
- **6 Python tool categories** — backing implementation of MCP wrappers
- **3 marketing frameworks** — Brunson Daily Seinfeld, Hormozi Value Equation, Channel Junkies hooks (paraphrased only)
- **Workflows** — 1-video-to-5-assets multiplier, voice-cloning pipeline, lead-magnet generation
- **Setup + troubleshooting** — installation, common MDX gotchas, glossary
